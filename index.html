v3.0.0 May 16, 2026 Major Update

Inline Editing, Pomodoro Breaks & More

Tasks are now fully editable from the detail sheet. The focus timer gains a proper Pomodoro break cycle with session history. Calendar and task management get powerful new capabilities.

  • Inline task editing — Tap date, time, priority, or category in the task detail sheet to edit them directly
  • Pomodoro break cycle — Automatic 5-minute short breaks and 15-minute long breaks every 4 sessions
  • Focus session history — Track completed focus sessions with today's stats and a full history log
  • Task sort options — Sort by priority, date, or name from the new sort button in the tasks header
  • Add tasks from calendar — "+" button on any calendar day pre-fills the date for quick task creation
  • Calendar task details — Tap any task in the calendar to open the full detail sheet with actions
  • Swipe actions on tasks — Swipe right to complete, left to delete with visual hints
  • Confetti celebrations — Particle animation when completing tasks
  • Skeleton loading — Shimmer placeholders while content loads
  • Shared element transitions — Notes expand smoothly into the editor
  • Accent color customization — 7 theme colors to choose from in settings
  • Note folders & drag-to-reorder — Organize notes into folders, reorder by dragging
  • Export to PDF/CSV — Export tasks as spreadsheet or formatted PDF
  • Progress & quote widgets — Home screen widgets for stats and daily motivation
  • Streak tracker — Daily completion streak displayed on the home screen
  • 6 bug fixes — Overdue stat accuracy, accent color consistency, timer duration, and more
